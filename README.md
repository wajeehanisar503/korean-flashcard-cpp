# korean-flashcard-cpp
A Korean vocabulary flashcard quiz app built in C++
A console-based Korean vocabulary flashcard quiz app built in C++.
Created by a self-directed Korean language learner pursuing the Global Korea Scholarship (GKS).
📖 About This Project
I built this app to combine two of my passions — programming and Korean language learning.
I am currently self-studying Korean through the King Sejong Institute curriculum (Level 4 completed) and preparing for the TOPIK II exam. Instead of using a generic flashcard app, I decided to build my own using C++ — turning my language learning journey into a coding project.
✨ Features
50 Korean vocabulary words across 8 categories
Interactive quiz — see the Korean word, type the English meaning
Instant feedback — Correct or Wrong with the right answer shown
Final score display at the end
