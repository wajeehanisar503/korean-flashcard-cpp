# 🇰🇷 Korean Flashcard Quiz App

A console-based Korean vocabulary quiz app built in C++.
Created by a self-directed Korean language learner pursuing the Global Korea Scholarship (GKS).

#📖 About This Project

I built this app to combine two passions — programming and Korean language learning. I am self-studying Korean through King Sejong Institute (Level 4 completed) and preparing for TOPIK II. Instead of using a generic app, I built my own in C++.
# ✨ Features

- 50 Korean vocabulary words across 8 categories
- Interactive quiz see the Korean word, type the English meaning
- Instant feedback  Correct or Wrong with the right answer shown
- Final score display at the end
